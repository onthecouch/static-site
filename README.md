# snawf.my.id

Source code for my personal landing page. It serves as a digital profile and hub for my self-hosted services and documentation.

## Overview

This is a static site built to be lightweight and simple. The design is inspired by terminal interfaces and dark mode aesthetics.

## Tech Stack

- **HTML5**: Semantic markup.
- **CSS3**: Custom styling using CSS variables, Flexbox, and CSS Grid. No frameworks.
- **Font**: IBM Plex Mono via Google Fonts.
