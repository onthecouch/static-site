Ashbloom Palette

Dark foundation with blooming color accents

Hey there. This palette started from my HD logo colors - those little accents I've been using forever in my personal branding. After staring at dark terminals for hours, I wanted something that felt like mine but still kept that cozy, focused vibe.

Slate is the base - it's that comfortable darkness where everything just works. Like a quiet server room at 2 AM. Gold is the main accent because, honestly, it just looks good against dark backgrounds and reminds me of old terminal amber displays.

The other colors came from experimenting. Purple felt right for secondary accents - deep enough to not clash, but still pops. Pink and Rose are my "hey, pay attention to this" colors for warnings and highlights. And Sky blue? That's my go-to for links and active states. It just feels clickable.

I call it Ashbloom because that's what it feels like - dark ash with little colorful blooms breaking through. Use it for terminals, dashboards, or anything that needs to be readable at 3 AM when you're debugging something.

Inspired by the colors in my HD logo. No fancy color theory here - just what looked good after too many late nights tweaking hex values.

How to Use

/* Include the palette */
<link rel="stylesheet" href="ashbloom.css">

/* Use CSS variables */
.my-element {
    background-color: var(--ashbloom-slate);
    color: var(--ashbloom-text-light); /* Ashbloom Cream */
    border: 1px solid var(--ashbloom-border);
}

/* Or use utility classes */
<div class="ashbloom-bg-gold ashbloom-text-slate">
    Gold background, dark text
</div>

/* Text color utilities */
<p class="ashbloom-text-light">Ashbloom Cream text</p>
<p class="ashbloom-text-muted">Muted text</p>
